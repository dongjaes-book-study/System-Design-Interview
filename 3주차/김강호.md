# 9장 웹크롤러설계

### 웹 크롤러 이용 사례
- 검색 엔진 인덱싱
- 웹 아카이빙
- 웹 마이닝
- 웹 모니터링

### 웹 크롤러 작업 흐름
1. URL 큐에 시작 URL 추가
2. URL 큐에서 URL 가져오기
3. URL 접속하여 웹 페이지 내용 다운로드
4. 웹 페이지 내용 파싱하여 올바른 형식을 갖춘 페이지인지 검증
5. 중복 컨텐츠 확인
   - 이미 저장소에 있는 콘텐츠인 경우 return
   - 저장소에 없는 콘텐츠인 경우 저장소에 저장 후 URL 추출기로 전달


### BFS vs DFS
- 웹 크롤러의 경우 보통 BFS 사용
